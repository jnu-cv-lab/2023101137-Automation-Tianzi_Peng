# 一、实验内容与要求  
#### 1.图像读入与预处理  
#### 2.下采样  
将原图缩小为原来的 1/2或1/4。缩小时建议分别测试:不做预滤波直接缩小，先做高斯平滑再缩小  
#### 3.图像恢复  
将缩小后的图像恢复到原始尺寸，分别使用以下方法:最近邻内插，双线性内插，双三次内插  
#### 4.空间域比较  
对原图与恢复图进行比较，至少完成:显示原图、缩小图、恢复图，计算MSE，计算PSNR  
#### 5.傅里叶变换分析  
分别对下列图像计算二维傅里叶变换并显示频谱:原图，缩小后图像，用双线性恢复后的图像  
要求:将频谱中心移动到图像中心，对幅度谱取对数显示，比较它们的高频成分差异，并说明原因  
#### 6.DCT 分析  
分别对原图和恢复图做二维 DCT:显示 DCT系数图(可取对数显示)，统计左上角低频区域能量占总能量的比例，比较不同恢复方法下 DCT 能量分布差是并给出解释  
# 二、实验结果分析  
- NEAREST: MSE = 20.9475, PSNR = 28.2170 dB  
- LINEAR: MSE = 16.2220, PSNR = 30.6411 dB  
- CUBIC: MSE = 12.0679, PSNR = 31.9847 dB  
从mse和psnr可以看出，恢复的效果CUBIC>LINEAR>NEAREST  
与原图相比，下采样图的幅度谱的四周变暗得稍微明显一点，而LINEAR恢复图的幅度谱的四周变暗得很快很明显，说明经过LINEAR后高频变少，图像比较平滑或模糊  
- Img low-frequency energy ratio: 0.9980  
- LINEAR_up low-frequency energy ratio: 0.9999  
- NEAREST_up low-frequency energy ratio: 0.9968  
- CUBIC_up low-frequency energy ratio: 0.9999  
原图和各恢复图的低频能量占总能量的比例都很接近1，说明图像的能量高度集中在低频，图像平滑、细节少、纹理弱，可压缩性好（JPEG 等可以丢掉更多高频系数而不会明显失真）  
NEAREST恢复图比原图的DCT图亮度衰减更快，说明高频成分更少，图像平滑、可压缩性好
