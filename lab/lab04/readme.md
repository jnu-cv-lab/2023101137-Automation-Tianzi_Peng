# 理论补充  
### 第一部分：下采样、混叠与高斯抗混叠  
- 什么是混叠（Aliasing）？ 想象车轮在电影里看起来像在倒转，这就是时间上的混叠。在图像中，当下采样（比如缩小图片）时，我们在减少像素的数量。如果图像中包含了非常密集的细节（高频信息），而我们采样的点不够密，这些高频细节就会“折叠”成低频的错误图案（比如锯齿状边缘或彩色的摩尔纹）。  
- Chirp 图与棋盘格： 这两种图是测试混叠的“神器”。Chirp 图从中心向外，条纹越来越密（频率越来越高）。直接下采样后，你会看到外圈原本密集的条纹变成了奇怪的粗条纹（这就是混叠）。  
- 如何消灭混叠？ 在下采样前加低通滤波（高斯滤波）。既然高频细节会导致混叠，我们就提前用高斯模糊把高频细节抹掉。
- FFT 频谱图： 傅里叶变换将图像从“空间域”转换到“频率域”。频谱图的中心是低频，越往外高频越多。对比滤波前后的频谱，可以清晰地看到高斯滤波把外围的高频信号咔嚓掉了，从而在频域上确认了混叠的消除。  
### 第二部分：验证 $\sigma$ 公式  
- 在下采样系数设为 $M=4$ 时，我们需要决定高斯模糊的强度，即标准差 $\sigma$。  
- 如果 $\sigma$ 太小（例如 $0.5$）： 模糊不够，高频信息没被完全切掉，下采样后依然会有混叠（摩尔纹残留）。如果 $\sigma$ 太大（例如 $4.0$）： 模糊过猛，不仅切掉了会导致混叠的高频，连正常的低中频结构也切掉了，下采样后图像会变得过度模糊，失去辨识度。  
- 理论最优值： 根据信号处理理论，为了完美匹配下采样率 M，高斯滤波器的截止频率应当对应新的奈奎斯特频率。推导得出的经验公式为 $\sigma \approx 0.45M$。  
### 第三部分：自适应下采样  
- 为什么要“自适应”？ 全局统一加高斯滤波有一个缺点：它对全图一视同仁。图像中的平滑区域（如天空）本来就没有高频，不需要模糊；而图像的边缘和密集纹理区（如草地）才需要抗混叠。统一模糊会让原本清晰的边缘也变糊。  
- 解决思路： 计算图像的梯度（Gradient）。梯度大的地方代表细节多、频率高，需要较大的 $\sigma$ 来抗混叠；梯度小的地方代表平滑，用极小的 $\sigma$ 甚至不模糊。由于 OpenCV 不支持逐像素设定不同 $\sigma$ 的高斯滤波，我们在代码中通常用图像融合（Alpha Blending）的技术来实现：基于梯度图作为权重，将清晰图和强模糊图进行按比例混合，再进行下采样。  

# 实验结果与分析  
### 第一部分：观察混叠与抗混叠及频域分析  
- Direct Downsample (Aliasing)（直接下采样）： 灾难现场。图像中心的宽圆环还算正常，但外围原本密集的细圆环，变成了一圈圈扭曲、粗大、甚至反向的奇怪条纹（这被称为摩尔纹）。这就是混叠（Aliasing）——因为采样点不够密，系统把高频的密集条纹“误认”成了低频的粗条纹。  
- Filtered Downsample (Anti-aliasing)（高斯滤波后下采样）： 图像外围没有了那些扭曲的丑陋条纹，取而代之的是一片平滑的灰色。虽然外围细节丢失了，但这是一种“安全”的丢失。它宁愿变成灰色（没有信息），也不输出错误的混叠信息。这就是抗混叠的意义。  
- FFT Original（原图频谱）： 傅里叶频谱图中心代表低频，越靠外代表高频。原图的频谱从中心向外都有亮度，说明原图包含了从低到高的丰富频率。  
- FFT after Gaussian（滤波后频谱）： 你会看到频谱图的边缘部分全黑了，只剩下中心一个亮圆圈。这就直观地证明了：高斯滤波器在下采样前提前把高频部分（会导致混叠的罪魁祸首）给剪掉了。  
### 第二部分：验证 $\sigma$ 公式
这个窗口有 4 张子图，展示了在下采样倍数 $M=4$ 的情况下，不同模糊强度带来的不同后果：  
- $\sigma = 0.5$ (Under-smoothed / 欠平滑)： 模糊力度太弱。频谱没剪干净高频。所以依然会在图像外圈看到明显的扭曲条纹（混叠残留）。  
- $\sigma = 1.0$： 混叠情况比 0.5 好一点，但在最边缘处依然能看到不自然的条纹。  
- $\sigma = 1.8$ (Optimal / 最优值)： 这是通过公式 $\sigma \approx 0.45M$ 算出的理论最优值。图像外围的高频区域刚好平缓过渡为灰色，几乎没有混叠，同时中心的低频结构也保持得相对清晰。这是保留信息与消除混叠的最佳平衡。  
- $\sigma = 4.0$ (Over-smoothed / 过度平滑)： 模糊力度太猛。不仅剪掉了高频，连中低频也剪断了。结果就是整张图看起来非常糊，连中心原本应该清晰的宽圆环也变得模糊不清了。  
### 第三部分：自适应下采样  
- 我认为棋盘图和Chirp图并不适合用来做统一下采样和自适应下采样的对比，因为整张图的绝大部分都已经是需要被过滤掉的高频信息，所以我自己生成了一张图，左半边是平滑的，右半边是高频的，这样就能使自适应结果在保持整体锐度的同时，实现了抗混叠的目标。  
- Gradient mask（局部梯度掩码图 / 热力图）：这是一张彩色热力图。红色代表这里细节多、频率高，需要强烈的模糊来抗混叠；蓝色代表这里很平坦，不需要模糊。  
- Uniform Downsampling和Adaptive (Gradient) Downsampling虽然在肉眼看区别不是很大，但是放大看局部图Center Detail就可以看到右边自适应下采样的边缘更加锐利，左边的统一下采样更模糊。
