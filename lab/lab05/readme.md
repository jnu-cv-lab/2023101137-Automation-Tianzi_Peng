# 实验要求  
已知一张包含矩形、圆、平行线、垂直线的测试图，分别对其施加：相似变换、仿射变换、透视变换，观察并总结每类变换对以下几何性质的影响  
1.直线是否保持为直线  
2.平行线是否仍保持平行  
3.两条垂直线是否仍垂直  
4.圆是否仍保持为圆  
最后，对一张存在逃视略变的平面图像进行校正(自己拍一张放在桌上的带文字和表格的A4纸)，看看自己是否满意，矫正的文字是否变形  
可以允许使用cv2.warpAffine、cv2.warpPergRective  
必须百己做：自己构造变柜阵、自己设计测试图、自己做几何性质判断、自己写结论  

# 实验结果  
根据实验程序的运行与视觉观察，三类变换对几何性质的保持情况汇总如下：  

| 几何性质 | 相似变换 (Similarity) | 仿射变换 (Affine) | 透视变换 (Perspective) |
| :--- | :--- | :--- | :--- |
| **直线是否保持为直线** | ✅ 保持 | ✅ 保持 | ✅ 保持 |
| **平行线是否仍保持平行** | ✅ 保持 | ✅ 保持 | ❌ 不保持（相交于消失点） |
| **两条垂直线是否仍垂直** | ✅ 保持 | ❌ 不保持（角度发生改变） | ❌ 不保持（角度发生改变） |
| **圆是否仍保持为圆** | ✅ 保持（仅半径改变） | ❌ 不保持（变为椭圆） | ❌ 不保持（变为椭圆或圆锥曲线）|

相似变换是**保角变换**，能够完全保留图形的形状特征，仅改变其大小、方向和位置。直线、平行、垂直和圆形性质均被完美保留。  
仿射变换保留了**共线性**与**平行性**，但破坏了角度和距离的比例。因此，垂直线不再垂直（发生倾斜），圆被拉伸成了椭圆。  
透视变换破坏了除**共线性**之外的所有几何不变性。平行线发生汇聚，垂直线夹角改变，圆发生严重畸变。  

在 A4 纸的校正实验中，通过提取畸变图像中纸张的四个角点，并将其映射到A4标准长宽比的矩形坐标上，成功用cv.getPerspectiveTransform和手动计算求解出变换矩阵M和H，可以看到求解结果M=H。将该矩阵应用于原图进行变换后，由于透视效应导致的字体“近大远小”及排版倾斜被完全消除，文字恢复了正常的视觉比例和正交排版，进一步验证了透视变换在图像几何平面映射中的有效性。
